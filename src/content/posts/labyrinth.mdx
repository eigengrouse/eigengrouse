The initial inspiration for this game came from [Super 48k Bro](https://rothers.itch.io/super-48k-bro) which simulates very fast horizontal scrolling on the ZX Spectrum by using pipes. That is, filling the screen with 8x8 character cells that are filled on one side so that `INK` and `PAPER` give you tall pixels that are very fast to manipulate. It gave me the idea of also using bars for a top-down view of a "dungeon" giving 4-way smooth scrolling using the same technique.

I used [z88dk](https://github.com/z88dk/z88dk) for this and iterated on the engine, and evolved a game out of that. A bit like how evolution seems to [converge into crabs](https://en.wikipedia.org/wiki/Carcinisation), this ended up becoming a [Sokoban variant](https://en.wikipedia.org/wiki/Sokoban). I created my own Z80 [graphics routines](https://github.com/eigengrouse/dungeon/blob/main/gfx/gfx.asm), and my own [AY tracker](https://github.com/eigengrouse/dungeon/blob/main/sfx/tracker.asm), and transcribed [Clotho](https://www.youtube.com/watch?v=admQ2Ideh0M) using [Sonic Pi](https://sonic-pi.net/) for the music, which I thought fit the strange ambience.
